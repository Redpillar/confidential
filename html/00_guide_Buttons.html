Primary Button

· 화면에서 가장 핵심적인 역할을 수행(저장,완료 등)하는 버튼
· 한 화면에 하나의 Primary를 권장
· Primary버튼은 다른 버튼과 시각적으로 명확하게 구별될 수 있도록 구성
· 버튼은 우정렬을 기본으로 하며 Basic버튼과 혼용 시 Primary Button을 최우측으로 배치

  • e.g. normal hover/focused readonly disabled

Basic Button

· 화면에서 기본 역할을 수행 (e.g. 삭제, 목록, 이전 등)하는 버튼
· 한 화면에 여러 개의 Basic 버튼이 혼용될 수 있다.
· Icon이 있는 타입과 없는 타입으로 나뉜다.
· Primary 버튼과 혼용시에는 아이콘을 적용하지 않는다.
· 예외적용) Refresh/Excel 등의 일부 기능버튼은 예외로 한다.
